Создайте веб-приложение для поиска книг по названию или автору с использованием Open Library API.
Ссылка на API: https://openlibrary.org/search.json?q=harry+potter
Пользователь вводит запрос (название книги или автора). Приложение отправляет его через библиотеку Axios. Для каждой книги показать:
⦁	Title 
⦁	Author 
⦁	First publish year 
⦁	Number of pages (если есть)
Во время загрузки используйте Bootstrap loader. Если книг по запросу не найдено, отобразите сообщение – «No results found».
При клике на книгу: открывайте модальное окно (Bootstrap) и покажите расширенную информацию (язык, ISBN).
